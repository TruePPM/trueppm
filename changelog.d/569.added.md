Added `Project → Settings → Integrations` page — a read-only summary of the project's outbound webhooks and inbound API tokens served by a new `GET /api/v1/projects/<id>/integrations-summary/` aggregator. The two static workspace stubs (`WorkspaceIntegrationsPage`, `WorkspaceWebhooksPage`) were misclassified per ADR-0076 and have been removed from the OSS bundle; the workspace routes `/settings/integrations` and `/settings/webhooks` now render a project-picker redirect shim so existing bookmarks keep working. The `project_settings.integrations` and `task_detail.external_links` slots were added to the widget registry as Enterprise extension points (per ADR-0029).
