- **Schedule build mode toggle in Settings**: build mode (the keyboard-first
  Schedule plan-entry surface) now has an in-app enable path. A new
  **Settings → Schedule → Build mode (beta)** toggle flips the
  `schedule_build_mode_v1` flag per browser, with no URL parameter, devtools, or
  rebuild required — the previous only ways to reach it. When on, a
  **View keyboard shortcuts** link opens the same cheatsheet you get from `?` on
  the Schedule view. The toggle is a local preference, so it applies instantly
  and does not touch the settings save bar.
