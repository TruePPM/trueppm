Project Monte Carlo runs are now persisted with a forecast history: each run records its P50/P80/P95 finish dates so you can see how a project's finish-date confidence drifts over time. The Monte Carlo drawer shows past runs newest-first with a per-run delta (e.g. "P80 +14d"), and the latest result now survives past the 24h cache. Run-author attribution is visible only to project Admins/Owners. OSS keeps the newest 100 runs per project (ADR-0109).
